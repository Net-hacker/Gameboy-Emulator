# Gameboy-Emulator
Ein von mir in C programmierter Gameboy-Emulator basierend auf Raylib

## Was muss noch getan werden?
- CPU
- Interpretieren
- Graphics
- Sound
- File Check

## Was funktioniert schon?
- Bootscreen

## Was brauche ich zum kompilieren
Entweder
`gcc`
oder
`clang`

## Und dann?

### Debug

#### gcc
`make debug_gcc`

#### clang
`make debug_clang`

### Release

#### gcc
`make release_gcc`

#### clang
`make release_clang`

## Wie kann ich es ausführen

### Debug
`./build/Debug/GB + eine .gb Datei`

### Release
`./build/Release/GB + eine .gb Datei`
